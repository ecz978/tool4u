Come funziona la codifica Base64

Base64 è un sistema di codifica che converte dati binari in testo ASCII utilizzando un set di 64 caratteri. È ampiamente utilizzato per trasmettere dati attraverso protocolli che gestiscono solo testo.

Quando utilizzare Base64

  • Email: Per allegare file binari in formato testo
  • Web Development: Per incorporare immagini in CSS/HTML
  • API: Per trasmettere dati binari via JSON
  • URL: Per codificare parametri con caratteri speciali
  • Database: Per memorizzare dati binari in campi testo

Caratteristiche del Base64

  • Utilizza 64 caratteri: A-Z, a-z, 0-9, +, /
  • Padding con "=" per completare gruppi di 4 caratteri
  • Aumenta la dimensione del dato del ~33%
  • È reversibile: puoi sempre decodificare
  • Non è crittografia: è solo una codifica

Esempi pratici

Testo: "Ciao mondo!" → Base64: "Q2lhbyBtb25kbyE="

Base64: "SGVsbG8gV29ybGQ=" → Testo: "Hello World"
